# Send SiteTracking data from the backend

- Doc ID: 122
- Source URL: https://mail.mailvio.com/apidocs/122
- Parent ID: 121
- Requires Auth: Yes
- Request Method: post
- Request URL: /accountmaster/tracker/transaction
- Published: Yes
- Last Updated: 2026-02-28T10:47:36.000Z

## Body

<p>While it is possible to manually send SiteTracking data from the client side via javascript, sometimes you might want to send SiteTracking data from an API or backend service.<br />This API route allows you to send data directly into SiteTracking from any backend.</p>

## Request Fields


```json
[
  {
    "fieldName": "uuid",
    "dataType": "GUID",
    "required": "on",
    "description": "The uuid that the client side javascript creates. Please only send a GUID that is generated by the client side javascript."
  },
  {
    "fieldName": "emailAddress",
    "dataType": "STRING",
    "required": "on",
    "description": "If a visitor has been identified, send the email address along with a valid UUID."
  },
  {
    "fieldName": "host",
    "dataType": "STRING",
    "required": "on",
    "description": "The URL of the website the visitor is visiting"
  },
  {
    "fieldName": "origin",
    "dataType": "STRING",
    "required": "on",
    "description": "The full origin of the website the visitor is visiting"
  },
  {
    "fieldName": "ip",
    "dataType": "STRING",
    "required": "",
    "description": "The IP Address of the visitor. If not supplied we will use the IP we detect from the server sending the request"
  },
  {
    "fieldName": "agent",
    "dataType": "STRING",
    "required": "",
    "description": "The user agent. If not supplied we'll use the sending servers user agent. Valid values [chrome,firefox,mobile_safari,mozilla,opera,safari,webkit,android]"
  },
  {
    "fieldName": "device",
    "dataType": "STRING",
    "required": "",
    "description": "The user device type. If not supplied we'll use the sending server device type. Valid values [desktop,phone,tablet,tv,bot,car]"
  },
  {
    "fieldName": "pathname",
    "dataType": "STRING",
    "description": "The path on your website the user is visiting"
  },
  {
    "fieldName": "query",
    "dataType": "STRING",
    "description": "Any query string the user is visiting"
  },
  {
    "fieldName": "customFields",
    "dataType": "OBJECT",
    "description": "An object of custom field names and values to enrich a contact if supplying emailAddress"
  },
  {
    "fieldName": "referrer",
    "dataType": "STRING",
    "description": "The referring website if available"
  },
  {
    "fieldName": "eventName",
    "dataType": "STRING",
    "description": "Used for notifying an event has been triggered. You can use system event types of Registration and Payment. Or anything else!"
  },
  {
    "fieldName": "eventData",
    "dataType": "OBJECT",
    "description": "An object of values that are associated with your event. If using a payment event you can pass {value: xxx.xx} for the amount paid. Otherwise supply anything here."
  },
  {
    "fieldName": "groupId",
    "dataType": "INTEGER",
    "description": "If a new contact is created, you can supply a group id here where we will create the new contact."
  }
]
```
