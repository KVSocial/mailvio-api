## Body

<p>Create Campaign Reports to retrieve lists of opens, sends, clicks, bounces and more.<br /><br />Reports are run in the background, and will be processed shortly after creation.<br />Once run, a file can be retrieved containing the results.<br /><br />In order to create a Campaign report, you would first create a report, retrieve the report it, poll the report ID until the report is complete, and finally request the file for the report.</p>
