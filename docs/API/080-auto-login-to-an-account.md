## Body

<p>If you have obtained an auth token by validating a login, or a registration, you can use the token to redirect users and have them seamlessly logged in.</p>
<p>Simply direct a user at the following url to auto log them in</p>
<div class="link-block">
<p>https://mail.mailvio.com/?usertoken=ACCESSTOKEN</p>
</div>
