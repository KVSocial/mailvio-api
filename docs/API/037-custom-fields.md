## Body

<p>A subscriber record only contains a few default fields. Email address, date created, parent groups, that kind of thing.</p>
<p>If you want to store more data against a subscriber record, we have created a custom fields system that enables you to add and manage as many custom fields as you need.</p>
<p>Custom fields are site wide, which means that if you don't need to create custom fields for each subscriber group. Instead you can create a custom field, and it can then be used on any subscriber in any group.</p>
