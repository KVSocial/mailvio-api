## Body

<h1>Introducing&nbsp;the Mailvio API</h1>
<h3>Comprehensive access to all of the power of the platform, from a simple API.</h3>
<p>The Mailvio Api exposes nearly every route that is used to power the entire Mailvio user interface. This means that if needed, developers are able to recreate almost any function or facility that Mailvio offers within their own apps.</p>
<p>So whether you need to build an entire email marketing platform of your own, or simply need to add a subscriber to a group when they register for your own app, you're covered.</p>
<h3>Quick Start Steps</h3>
<p>Before you can begin making API calls you will need to have an account, and a corresponding auth token to gain access. The following steps will get you up and running quickly...</p>
<ol>
<li><a href="/apidocs/2">Create an Account</a><br />Create your first account and retrieve an auth token which can be used for future requests.<br /><br /></li>
<li><a href="/apidocs/21">Create a Group</a><br />Now you have an auth token, you can use this guide to create your first group, which you can push subscribers into.<br />&nbsp;</li>
<li><a href="/apidocs/24">Add a Subscriber</a><br />You've got an account, you've created a group. Use this guide to send in your first subscriber.</li>
</ol>
